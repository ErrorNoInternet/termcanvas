# termcanvas
Draw stuff in your terminal!
![Screenshot](https://raw.githubusercontent.com/ErrorNoInternet/termcanvas/main/screenshots/0.png)

## Usage
`esc`: exit termcanvas

`left click`: place a pixel

`right click`: remove a pixel
